    • Unpacking a Tuple
      • In Python, values from within the tuple can be extracted out and assigned to variables.
      • The number of variables must match the number of Tuples.
        • If you want more than one item to belong to the variable, you must place a * in front of the variable.
        • If there is more than one set of data within the tuple, that has more than one element assigned to it,
          then the data must be unpacked together as one variable, and then break it down with lists.
          1. Assign the tuple data to variables:
            2. cost, teacher, bring_lunch, *students_and_assistants, bus_company = tuple_data
          2. Break down the combined tuple data into their own variables, using indexes 
            3. students = students_and_assistants[:4]
assistant_teachers = students_and_assistants[4:6]
          tuple_data = (50, "Mr. Smith", True, "John", "Alice", "Bob", "Emily", "Ms. Johnson", "Mr. Brown", "ABC Bus Company")

cost, teacher, bring_lunch, *students_and_assistants, bus_company = tuple_data

students = students_and_assistants[:4]
assistant_teachers = students_and_assistants[4:6]

print("Cost:", cost)
print("Teacher:", teacher)
print("Bring lunch:", bring_lunch)
print("Students:", students)
print("Assistant Teachers:", assistant_teachers)
print("Bus Company:", bus_company)
      myTuple = (2.358, "Robby", False, "John", "Lisa", "Li -Mae", "Carey", ("Onions, Potatoes"))
(cost, teacher, bring_lunch, *students, groceries) = myTuple

    f"Teacher = {teacher}\n"

    f"Students = {students}\n"

      Creating Objects with Super class

      Super Class:
      • An already defined class, that will be adopted by the current class.
      • Adds functionality of super class into current class.

        • Super Class 
        
class RobotIntro:
def __init__(self):
    self.name = ""

def intro(self):
    print(f"My name is Bond. {self.name} Bond!")
        Adding Super Class to current class 
        class RobotIntro:
    def __init__(self):
        self.name = ""

    def intro(self):
        print(f"My name is Bond. {self.name} Bond!")


class Robot4(RobotIntro):                   #Adding Super Class as an argument when building the class
    def __init__(self, name, weight):
        super().__init__()                  #Initializing the Super Class
        self.name = name
        self.weight = weight


r5 = Robot4("Kia", 87)
r5.intro()
        # Output:
My name is Bond. Kia Bond!
